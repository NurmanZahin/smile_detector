Model Architecture


This classification model uses a MobileNetV2 architecture as its backbone that has been pretrained on ImageNet.

Olivetti Faces Dataset


The model was trained on the Olivetti faces dataset. Training images were preprocessed using methods from MobileNetV2.

Training Results


Accuracy of 70% was achieved on the validation set. Adam optimizer and categorical cross entropy loss function was used.
